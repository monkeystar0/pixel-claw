# pixel-claw

Visualize OpenClaw agent sessions as interactive pixel art characters in a multi-room virtual office.

## Quick Start

```bash
npm install
npm run dev
```

## Configuration

See `.env.example` for all configuration options.

## Controls

- **WASD / Arrow Keys** - Move player character
- **E** - Interact with nearby agent or furniture
- **Escape** - Close interaction panel
- **?** - Show help overlay
